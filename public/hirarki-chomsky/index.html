Pada tahun 1956-1959 Noam Chomsky melakukan penggolongan tingkatan dalam bahasa berdasarkan aturan produksi,yaitu menjadi empat class yang disebut Hirarki Chomsky .
Penggolongan tingkatan yang dilakukan oleh Noam Chomsky ada 4 class yaitu :

1. Tipe 0 (Unrestricted)
Pada tipe 0 ini tidak ada batasan pada aturan produksi .
Contoh :
Ab -> Cde
Abc -> d
2. Tipe 1 (Context Sensitive)
Pada tipe 1 ini “panjang String ruas kiri harus lebih kecil atau sama dengan ruas kanan”.
Contoh :                           

  • AbC -> deF                          
  • A -> b

3. Tipe 2 (Context Free)
Ruas kiri harus tepat 1 simbol variabel,yaitu simbol non terminal
Contoh :                                    
A -> deF                                        
B -> Cd

4. Tipe 3 (Regular)
Ruas kanan hanya memiliki maksimal 1 simbol non terminal,dan di letakkan paling kanan.
Contoh                                         
A -> dfE                                          

Namun yang biasa dipakai adalah tipe ke 2(Context Free).Oke Segini dulu penjelasan tentang penggolongan tingkatan dalam bahasa oleh Noam Chomsky,Semoga bermanfaat dan maaf jika ada kesalahan bisa di ralat dan tulis di Komentar ^^.
