Cara Sharing Wifi,Membuat Tethering di Windows 8 tanpa software

January 22, 2015   

Assalamu’alaikum

Mungkin masih ada yang belum tau bagaimana caranya sharing wifi dari laptop/pc kita ,atau malah sudah pada tau caranya dengan menggunakan software tapi softwarenya itu sumber virus.Nah kebetulan saya akan sharing sedikit pengetahuan tentang Cara Sharing Wifi di laptop tapi tanpa software yang konon itu banyak virusnya,bukannya menyelesaikan masalah malah bikin rudet hehe.Berhubung ini Tutorial Perdana saya dan buatnya juga jam setengah dua malam jadi ngantuk,mohon maaf kalau ada kesalahan atau ada yang norak haha.
Oke Yuk mari kita ke pembahasan :.

1. Pertama Klik Kanan di Start Menu yang berlambang Windows





2. Selanjutnya kita buka cmd(Command Prompt)sebagai Administrator,dengan 
KLIK–>Command Prompt(Admin)



3. Setelah itu akan muncul cmdnya ,lalu ketik di cmdnya   netsh wlan set hostednetwork mode=allow ssid=NamaNetwork key=passwordnya.
lalu tekan Enter.


Sampe tahap ini network telah terbentuk,tandanya berhasil adalah seperti ini


Tapi sampe sini belum bisa di gunakan karena belum kita jalankan karena baru kita buat saja.

4. Untuk Menjalankannya dengan mengetikkan   netsh wlan start hostednetwork,lalu tekan Enter


5. Selesai deh sudah bisa di pakai ,tapi jika ingin stop sharing wifinya bisa juga dengan mengetikkan 
 netsh wlan stop hostednetwork

Selesai sudah pembahasan tentang cara sharing wifi/membuat tethering di Windows 8.
Semoga Bermanfaat.
